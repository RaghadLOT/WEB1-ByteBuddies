Courses



"Python is fun"

Embark on a fun and exciting journey into the world of coding with our "Python is fun" course designed specifically for kids! This course introduces the basics of programming using Python, a language known for its simplicity and versatility

Game programming 1 with Python

In this beginner level Python Programming course, students will create a fun 2D game from the ground up. Students will build the entire game from scratch using Python code and learn about functions, loops, collision detection, game logic and more!

Game Programming 1 With Javascript

In this beginner level JavaScript programming course, kids build a 2D Infinite Runner style game from the ground up! We cover key concepts such as conditionals, loops, variables, and more!

Game Programming 2 With Javascript

In this Intermediate level Javascript coding course, kids will build an exciting retro Tower Defense game from the ground up. We cover key concepts such as conditionals, loops, collision detection, enemy pathing, vectors, and more.

Scratch Catch the Cat!

In this beginner Scratch coding course, students will learn to make a fun game where the player scores points by clicking on a fast, very sneaky cat who doesn't want to get caught! We cover key concepts such as loops and conditionals.

Scratch Fruit Slash

In this beginner Scratch coding course, kids learn to make a 'Fruit Ninja' style game where the player has to slash as much fruit as possible while avoiding enemies. We deepen our understanding of concepts such as variables, functions, loops and conditionals.
